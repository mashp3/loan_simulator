# ローンシミュレータ / Loan Simulator

住宅ローンや自動車ローンの返済計画を簡単にシミュレーションできるアプリです。

## 機能 / Features

- ローン計算（元利均等返済）
- ボーナス返済シミュレーション
- 繰上返済効果の計算
- 複数プランの比較（プレミアム機能）
- 借入診断機能（プレミアム機能）
- 返済スケジュールのCSV出力（プレミアム機能）

### よくある質問 / FAQ

**Q: プレミアム機能が購入できません**  
A: インターネット接続を確認し、App Store または Google Play にサインインしていることをご確認ください。

**Q: 購入したのにプレミアム機能が使えません**  
A: アプリを再起動してください。それでも解決しない場合は「購入を復元」をお試しください。

**Q: 計算結果を保存できません**  
A: データ保存機能はプレミアム版（¥230）の機能です。
